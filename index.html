In this practicum you will develop a simple responsive web application in React. You need to create a new repository and name is imagequiz. Do NOT use your ISTA330 repository for your development and deployment of this practicum. You need to create a separate repository for this practicum.

The web application should have the following pages:

  1. A registeration (sign up) page
  2. A login page
  3. A home page
The file data.js contains several images. The homepage of the application should list all the images. For each image it should show the image and under the image it should display the name of the image. Your web application should be responsive. So you should use React Bootstrap (https://react-bootstrap.github.io) components.
Deploy your react application to github pages. If deployed successfully then your react application will be shown in the iframe below.


Follow the following instructions to set up the react application and deploy it to Github Pages:

  1. On Github, create a repository and name it imagequiz.
  2. Clone the repository on your local machine. Use the command: git clone https://github.com/yourusername/imagequiz.git
  3. On terminal or command prompt, change the current directory to the folder imagequiz created by the command at step 2 and run the following command (you need to have installed Node):
    npm init react-app imagequiz
  4. Install GitHub Pages package as a dev-dependency: first run the following: cd imagequiz and then run npm install gh-pages --save-dev
  5. Now open the imagequiz folder in your visual studio code and locate the file package.json. Add the following line to the file:
    "homepage": "http://yourusername.github.io/imagequiz"
  6. In the existing scripts property in the file package.json add the following:
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build",  
  7. Commit and push your changes to Github using your Visual Studio Code.
  8. Deploy your project to Github Pages by running the following command:
    npm run deploy
